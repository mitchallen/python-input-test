python-input-test
==

Author: Mitch Allen


## Usage

* Requires pyhon 3.10 or later
* Can run in a Microsoft Dev Container

```shell
python app.py
```

### Command aexmple

```shell
:> add 5.0
:> add 20
:> sub 3
:> quit
```

## Testing

To run all tests:

```shell
python -m unittest
```

To run individual tests:

```shell
python test_calc.py
python test_app.py
```
